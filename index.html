유클리드 최소 거리 알고리즘 (성향 테스트용)

사용자가 수치(1~10)를 답하면, 이를 4차원 좌표 $(S_L, S_W, P_L, P_W)$에 투영합니다. 그 후 실제 iris.csv의 대표 품종별 중심점까지의 거리를 계산하여 가장 거리가 가까운 군집으로 매핑합니다.

$\text{Distance} = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2 + (x_3 - y_3)^2 + (x_4 - y_4)^2}$

의사결정 트리 알고리즘 (의사결정 모드용)

전체 데이터를 수차례 가르고 가르는 반복 스무고개 모델입니다. 각 지점의 다른 클래스들이 섞여 있는 비율을 정형화하는 지니 불순도(Gini Impurity)가 낮아지는 임계점들을 컴퓨터가 분석하여 모델의 기틀을 잡습니다.

$\text{Gini Impurity} = 1 - \sum (P_i)^2$
